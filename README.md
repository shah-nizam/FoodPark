# FoodPark
ICT2103 - Information Management Project

A web application that allows people to browse Food Establishment and their nearest Carpark.

Technologies used:
1. PHP
2. Javascript
3. MySQL
4. Google API
5. LTA Datamall API

# How to configure

1. Import foodfinderapp.sql to MySQL
2. Change credentials at protected/databaseconnection.php to connect to MySQL database

# Things to note

I used Xammp to configure it locally

Google and LTA Datamall API keys are removed

Email credentials are removed

Please add your own key and credentials
